# Gestión de Datos de Países en Python

## Integrantes
- Valentino Altamira
- Angel Sanchez

## Descripción
Programa desarrollado en Python para gestionar información de países utilizando listas, diccionarios, funciones y archivos CSV.

## Funcionalidades
- Agregar países
- Actualizar población y superficie
- Buscar países
- Filtrar por continente
- Filtrar por rango de población
- Filtrar por rango de superficie
- Ordenar países
- Mostrar estadísticas

## Archivos del proyecto
- tpi_prog.py
- paises.csv

## Ejecución

```bash
python tpi_prog.py
```
